Readme

Run Pathfinding as Startup Project

Hotkeys:

F1: Set search mode to Breadth First
F2: Set search mode to Depth First
F3: Set search mode to Dijkstra
F4: Set Search Mode To A_Star

F5: Toggle Search Pattern visualization


Right Click: Set new destination.  

Numeric Keys 1-3: Set Tile Type (1:Grass, 2: Sand, 3:Water)
Numeric Key 0: Set tile Type to impassable obstacles

Shift + Left-Click: Paint tile with current tile Type
